Multi-Condition Diagnosis Using Vital and Image

  • Tech Stack: Python (FASTapi Frameworks), HTML, CSS, JS, Scikit-learn, Pandas, NumPy, Matplotlib
  • Github URL: Project Link

The system allows the admin to register and log in securely to access the dashboard. After login, the admin can use the menu bar to navigate through modules such as analytics, diagnosis results and data upload. The dashboard displays analytics of previously attended cases and diagnosis history using charts and reports. The admin can upload patient or dataset information for diagnosis prediction and the system processes the data to generate results. Finally, the diagnosis results are stored both in the system storage and database for future reference, reporting and analytics.

Developed an intelligent healthcare prediction system to identify risks of multiple diseases using patient medical records, symptoms and vital parameters. Collected, cleaned and preprocessed healthcare datasets by handling missing values, encoding categorical data and normalizing features for improved model performance. Performed Exploratory Data Analysis (EDA) using statistical methods and visualizations to identify patterns, correlations and disease trends in patient data.

Implemented feature selection techniques to identify the most influential medical attributes affecting disease prediction accuracy. Built and trained machine learning classification models using Scikit-learn algorithms such as Logistic Regression, Decision Tree and Random Forest. Evaluated model performance using accuracy score.

Designed an automated diagnostic reporting module that generates prediction summaries with confidence scores for better clinical interpretation. Improved prediction reliability by comparing multiple models and selecting the best-performing algorithm based on evaluation metrics. Used Pandas and NumPy for efficient data manipulation, transformation and analysis of large healthcare datasets. Created data visualizations and performance graphs using Matplotlib to present insights and model evaluation results clearly.

Developed a scalable and modular project structure to support future integration of additional disease prediction models. Enhanced understanding of machine learning workflow including preprocessing, training, testing, validation and deployment concepts.
