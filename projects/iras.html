News Website Scraper

  • Tech Stack: Python, BeautifulSoup, Pandas, JSON, Requests
  • Github URL: Project Link

Developed a web scraping and data extraction pipeline to collect and process large-scale news article data from online news platforms including The Times of India. Scraped and processed news articles by extracting headlines, publication dates, article categories, author details and article content. Utilized Python libraries such as BeautifulSoup and Requests to parse HTML pages and efficiently retrieve structured information from websites.

Implemented data cleaning and preprocessing techniques to remove duplicates, handle missing values and normalize extracted text data. Converted unstructured web data into structured JSON and tabular formats for easier storage, querying and downstream analytics. Performed keyword frequency analysis and trend identification using Pandas to derive insights from news datasets.

Built automated batch extraction workflows to support continuous and scheduled data collection processes. Designed reusable and modular scraping scripts to improve maintainability and scalability for multiple news sources. Stored processed datasets in CSV/JSON formats for use in data visualization, NLP tasks and analytical dashboards.
